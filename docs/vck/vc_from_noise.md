### Modifier Parameters {: data-toc-skip }
- **`Noise Scale`**: Controls the scale of the base noise octave.
- **`Noise Detail`**: Determines the number of noise octaves. It can accept fractional values, which results in a blend between the two integer values.
- **`Noise Roughness`**: Allows you to blend between a smoother noise pattern and a rougher one with sharper peaks.
- **`Noise Lacunarity`**: Defines the difference in scale between consecutive octaves. Higher values result in a larger scale for higher octaves.
- **`Noise Distortion`**: Controls the amount of distortion applied to the noise.
- **`Noise Contrast`**: Adjusts the value range between light and dark areas.
