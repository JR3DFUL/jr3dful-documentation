### Modifier Parameters {: data-toc-skip }
- **`Attribute Float`**: Sets the source float attribute.
- **Pack to <span style="color: #E53935;">R</span>**: Writes the source float value into the Red channel.
- **Pack to <span style="color: #43A047;">G</span>**: Writes the source float value into the Green channel.
- **Pack to <span style="color: #1E88E5;">B</span>**: Writes the source float value into the Blue channel.
- **Pack to <span style="color: #9E9E9E;">A</span>**: Writes the source float value into the Alpha channel.

!!! info "Mix & Mask Parameters"
    Refer to the [Mix & Mask](./vc_mix_and_mask.md) page for a full description of these settings.
