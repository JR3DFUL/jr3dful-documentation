### Modifier Parameters {: data-toc-skip }
- **`UV0`**: Selects the first source UV Map.
- **`Pack UV0 To`**: Determines which channels the first UV map is packed into (e.g., RG for Red and Green).
- **`UV1`**: Selects the second source UV Map.
- **`Pack UV1 To`**: Determines which channels the second UV map is packed into (e.g., BA for Blue and Alpha).
