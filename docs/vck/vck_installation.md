# Installation

Vertex Color Kit is a collection of Geometry Node modifiers installed via Blender's Asset Library.

---
## Installation Steps

1.  **Download and Extract**
    Download the latest `VertexColorKit.zip` file. Extract the entire contents of the zip file to a permanent location on your computer where you plan to keep your assets.

2.  **Open Blender Preferences**
    Launch Blender and navigate to the preferences window by selecting `Edit > Preferences`.

3.  **Add Asset Library**
    In the Preferences window, go to the **File Paths** tab. Find the **Asset Libraries** section and click the **`+`** button to add a new library path.

4.  **Select the Folder**
    Blender will open a file browser. Navigate to the folder where you just extracted the `VertexColorKit` files, select it, and click the **Add Asset Library** button.

5.  **Save and Restart**
    Save your preferences by clicking the button in the bottom-left of the Preferences window. **You must restart Blender** for the asset library to be fully loaded.

## Accessing the Modifiers
After restarting Blender, open the **Asset Browser**. You will find all the modifiers under the **"Vertex Color Kit"** category. Simply drag and drop them from the Asset Browser onto your mesh object.

!!! info "Official Documentation"
    For more detailed information on how Blender's Asset Library works, you can refer to the [Official Blender Manual](https://docs.blender.org/manual/en/latest/files/asset_libraries/introduction.html).
