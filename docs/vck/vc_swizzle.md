### Modifier Parameters {: data-toc-skip }
- <strong>Change <span style="color: #E53935;">R</span> To</strong>: Remaps the source for the Red channel.
- <strong>Change <span style="color: #43A047;">G</span> To</strong>: Remaps the source for the Green channel.
- <strong>Change <span style="color: #1E88E5;">B</span> To</strong>: Remaps the source for the Blue channel.
- <strong>Change <span style="color: #9E9E9E;">A</span> To</strong>: Remaps the source for the Alpha channel.

!!! info "Mix & Mask Parameters"
    Refer to the [Mix & Mask](./vc_mix_and_mask.md) page for a full description of these settings.
